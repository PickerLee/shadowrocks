# shadowrocks

> 请确保系统已安装`node`程序
> 确保`ssstarter.sh`拥有可执行权限
```bash
chmod +x shadowrocks/ssstarter.sh
```

## Usage
```bash
./add-port-password.sh PORT PASSWORD
```
